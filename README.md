# myRoutine App

## Descripción

**myRoutine** es una aplicación mobile desarrollada en ReactNative fue pensada para solucionar los problemas de las personas que entrenan en los gimnasios; la misma nos permite crear rutinas de entrenamientos con un nivel de detalle único para no generarnos ningún tipo de duda a la hora de ejercitarnos.

### Funciones

#### Screens

- myRoutine: nos permite crear y ver nuestras rutinas, también podemos utilizar las que estan descargadas de manera predeterminada
- Entrenamiento: al momento de realizar la actividad física nos permitirá ver como y que cantidad de ejercicioos debemos hacer, también llevar el control de que ejercicios ya hemos realizado para estar mejor organizados
- Perfil: aquí podremos ver todos nuestros datos, a futuro habrá más novedades, ya sé compartir rutinas con otros usuarios, ver el historial de nuestra actividad, entre otros.
